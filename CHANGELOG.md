## [0.1.0] - 2025-03-26

### 新增

- 初始版本发布
- 多智能体并发推演引擎
- 8 种内置思维框架
  - 第一性原理 (First Principles)
  - 反脆弱 (Anti-Fragility)
  - 系统思维 (Systems Thinking)
  - 水平思维 (Lateral Thinking)
  - 行为经济学 (Behavioral Economics)
  - 等
- 结构化交叉质询流程
- 风险显式标注功能
- 行动计划生成模块
- 议题澄清对话界面
- 会话历史管理
- 多 LLM 供应商支持
- 本地配置持久化

### 技术栈

- 前端：React 19 + TypeScript + Tailwind CSS 4 + Vite
- 后端：Tauri 2 (Rust)
- 测试：Vitest + Playwright
