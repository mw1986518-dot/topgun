# TopGun

**顶级思维** - 一个基于多智能体并发推演的 AI 辅助决策工具。

## 项目目标

TopGun 旨在帮助用户对复杂议题进行深度思考和决策推演。通过模拟多个不同思维框架的"智能体"同时对同一议题进行分析、质疑、修正，最终达成共识或识别风险1。

核心能力：
- **议题澄清**：通过多轮追问，帮助用户厘清真正的问题本质
- **多视角推演**：不同思维框架从各自角度审视问题，发现盲点
- **异议与修正**：智能体之间相互质疑、补充，避免单一视角的局限
- **风险识别**：显式标注无法达成共识的风险点，辅助决策
- **落地方案**：将共识转化为可执行的行动计划

## 适用人群

- **决策者**：需要在复杂情境下做出关键判断的管理者
- **产品/项目经理**：规划产品方向、评估方案可行性
- **创业者**：验证商业想法、识别潜在风险
- **研究者**：梳理研究思路、发现逻辑漏洞
- **任何需要深度思考的人**：当你面对复杂问题，希望获得更全面的视角

## 技术栈

- **前端**: React 19 + TypeScript + Tailwind CSS 4 + Vite
- **后端**: Tauri 2 (Rust)
- **测试**: Vitest + Playwright

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run tauri dev

# 运行测试
npm test

# 构建
npm run tauri build
```
