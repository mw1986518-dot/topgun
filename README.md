# Idea Refinery

AI 认知提纯器 - 一个基于 Tauri 的桌面应用，用于与 AI 进行深度交互和思维提炼。

## 技术栈

- **前端**: React 19 + TypeScript + Tailwind CSS 4 + Vite
- **后端**: Tauri 2 (Rust)
- **测试**: Vitest + Playwright

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run tauri dev

# 运行测试
npm test

# 构建
npm run tauri build
```
